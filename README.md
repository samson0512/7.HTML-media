# 7.HTML-media
# program:
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Media Example</title>
</head>
<body>
    <h1>HTML Media Elements</h1>
    
    <section>
        <h2>Audio Player</h2>
        <p>This is an example of an audio player. You can play, pause, and adjust the volume using the controls provided.</p>
        <audio controls>
            <source src="path/to/your/audiofile.mp3" type="audio/mpeg">
            <source src="path/to/your/audiofile.ogg" type="audio/ogg">
            Your browser does not support the audio element.
        </audio>
    </section>
    
    <section>
        <h2>Video Player</h2>
        <p>This is an example of a video player. You can play, pause, and adjust the volume using the controls provided. You can also view the video in full-screen mode.</p>
        <video controls width="600">
            <source src="path/to/your/videofile.mp4" type="video/mp4">
            <source src="path/to/your/videofile.ogg" type="video/ogg">
            Your browser does not support the video tag.
        </video>
    </section>
</body>
</html>
```
